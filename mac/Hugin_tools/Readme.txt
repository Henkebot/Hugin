== How to "install" this package ==
- Copy the Hugin_tools directory inside this dmg to a location of your liking.


== How to use the tools inside the Hugin_tools directory ==
- Open a terminal
- cd into the Hugin_tools directory
- issue the command "source ./set_environment.txt"   (without the double quotes)

Now the tools inside this directory are set as first entries in your PATH statement.
Note: This is not a permanent change but only in this terminal session.
