Nona gui

nona_gui is a GUI version of the nona stitcher. It has all the same dialog boxes and progress bars as the Windows version of PTStitcher, so in principle it can be used as a drop-in replacement for PTStitcher when using PTGui, PTAssembler or one of the other Windows GUI front-ends.

nona_gui is distributed with hugin.
